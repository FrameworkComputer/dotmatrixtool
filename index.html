LED Matrix Input Module Control

Instructions

Left mouse button to draw.
Right mouse button (or ctrl+left) to erase.

Steps:

  1. Connect Module
  2. Click Connect Button
  3. Allow connection and select port
    (something like ACM0 or COM3)
  4. Start drawing
